# CLEAVE

```
cleave
```

The `cleave` command executes a sweeping attack that strikes all enemies within your weapon's optimal range simultaneously. This action requires a two-handed weapon.

*Example*
```
cleave
```

## Requirements
- Two-handed weapon equipped
- At least one enemy within optimal weapon range

## Mechanics
- Affects every hostile combatant at the weapon's optimal weapon range
- Separate attack rolls for each target
- All strikes occur simultaneously
