```text
# Establish neural connection with the Workbench
# Only works while the player's consciousness is "co-located" with the workbench object in the game world
> use workbench

# Arsenal management - consciousness surveys available shells
> shell list                    # Display all shells in the three-shell arsenal
> shell swap <shell_name>       # Transfer consciousness to different shell
> shell status                  # Comprehensive analysis of active shell
> shell reorder <from> <to>     # Change shell list order

# Shell optimization - direct stat manipulation
> shell stats                                 # Display current shell stats
> shell stats add POW|FIN|RES <increment>     # Allocate enhancement points
> shell stats remove POW|FIN|RES <increment>  # Redistribute capabilities

# Component integration - physical shell modification
> shell component list                        # Inventory mounted systems
> shell component examine <component_name>    # Detailed component analysis
> shell component mount <component_name>      # Install new hardware
> shell component unmount <component_name>    # Remove existing systems

# Inventory and resource management
> inventory components                        # List available components
> inventory materials                         # Show raw materials and resources

> shell commit                          # Finalize stat modifications; consumes resources
> shell undo                            # Revert pending stat changes

# Exit the Workbench
> exit|quit|bye
```
